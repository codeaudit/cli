usage: riseml [-h] [-v] [--version] {whoami,user,system,init,train,monitor,logs,kill,status} ...

positional arguments:
  {whoami,user,system,init,train,monitor,logs,kill,status}
    whoami              show currently logged in user
    user                modify users
    system              system level commands
    init                create config file for this directory
    train               run new experiment or experiment series
    monitor             show monitor
    logs                show logs
    kill                kill on-going experiment or experiment series
    status              show (running) experiments

optional arguments:
  -h, --help            show this help message and exit
  -v                    show endpoints
  --version, -V         show version
